Design #3 Key Concepts

Donut Chart Overview

Proportion-based status visualization. 50 readers' distribution is understood in under 1 second. Center shows total count. Legend doubles as filter buttons.

Waffle Dot Grid (10x5)

Each cell = 1 reader. Inspired by GitHub contribution graph. Color-coded background + animated ring indicator. 50 readers fit perfectly in a 10-column grid without scrolling.

Ring Status Indicator

Reading: Spinning ring + pulsing center dot.
Error: Dashed ring spinning reverse + fast blink.
Connected: Static solid ring.
Disconnected: Gray, subdued.

Minimal Information Hierarchy

Level 1: Color (instant status recognition).
Level 2: Ring animation (activity state).
Level 3: Number label (reader ID).
Level 4: Tooltip on hover (full details).

Swing Implementation

  • Donut chart: Graphics2D Arc2D.Double segments, anti-aliased. Center text with drawString.
  • Waffle grid: GridLayout(5, 10) with custom-painted JPanel cells.
  • Ring animation: javax.swing.Timer rotating arc start angle per frame.
  • Dashed ring (error): BasicStroke with dash array + rotating transform.
  • Tooltip: Override JToolTip or use JPopupMenu with HTML content.
  • Progressive disclosure: Color visible always, details on hover - reduces cognitive load.
  • Layout: BorderLayout(WEST=donut panel, CENTER=grid) inside the existing ReaderStatusPanel area.
