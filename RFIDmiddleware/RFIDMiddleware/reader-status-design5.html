Design #5 - Radial Ring Dashboard

Radar/Sonar-inspired layout. 50 readers arranged in two concentric rings around a central hub. Each node is a color-coded circle. The spatial arrangement makes it easy to scan all readers in a circular sweep, like a radar screen.

Key Elements

  • Center Hub: Shows aggregate stats by default. Click a reader node to show its details in the hub. Back button returns to overview.
  • Dual Rings: Outer ring (30 readers) + inner ring (20 readers). Nodes are spaced evenly.
  • Node Animation: Reading nodes pulse with glow. Error nodes blink rapidly. Disconnected nodes are dimmed.
  • Connector Lines: When a reader is selected, a line connects the node to the center hub.
  • Side Panel: Scrollable list for precise selection. Status filter chips. Search.

Swing Implementation

  • Ring layout: Calculate x,y with cos/sin in paintComponent. Two radii for dual rings.
  • Nodes: fillOval with color per status. Glow = semi-transparent larger oval behind.
  • Center hub: Overlapping JPanel with CardLayout (overview vs detail).
  • Connector line: Graphics2D.drawLine from node center to hub center with alpha stroke.
  • Mouse detection: Point distance check in mouseClicked/mouseMoved for circular hit testing.
  • Side panel: JScrollPane + BoxLayout of row JPanels. Click highlights node + shows detail in hub.
